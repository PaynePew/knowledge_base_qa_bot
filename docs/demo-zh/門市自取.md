# 門市自取

本文件說明 ACME 商店「線上購買、門市取貨」服務的適用範圍、取貨身分驗證、保留期限與通知方式等相關規定。

## 線上購買、門市取貨

顧客可於 ACME 商店線上購物，並選擇至實體門市取貨，免額外支付運費。操作步驟如下：

1. 瀏覽線上商品目錄，將欲購買的商品加入購物車。
2. 結帳時選擇「線上購買、門市取貨」選項。
3. 系統將依您填寫的郵遞區號，列出鄰近且有現貨的門市供您挑選，請選擇方便取貨的門市據點。
4. 完成付款後，您將收到一封電子郵件確認信，內含訂單明細與預估可取貨時間。

一般而言，現貨商品約於 **2 小時內** 完成備貨；若遇尖峰時段或大型訂單，備貨時間最長可能需要 **24 小時**。當您的訂單完成備貨、可供取貨時，系統將另行發送通知。

選擇門市自取 **不需額外支付任何費用**，是兼具便利與經濟的取貨方式。

## 不適用門市自取的商品

並非所有商品皆可選擇門市自取。下列因素可能導致商品無法以門市自取方式出貨：

- 商品體積過大。
- 商品庫存不足。
- 商品受運送條件限制。

凡屬上述情形的商品，將於該商品的商品頁面上標示為不適用門市自取，請於下單前留意頁面說明。

若您在下單後需要變更取貨門市，請儘速聯絡客服團隊，以利我們及時為您調整。

## 取貨身分驗證

為確保取貨流程安全、順暢，並保障您的權益，顧客取貨時須出示有效證件，以驗證取貨人身分、確保商品交付予正確的對象。請於抵達門市時備妥下列文件：

- 確認信（電子郵件確認信的紙本或電子畫面皆可）。
- 政府核發之有效附照片身分證件。

可接受的身分證件包括：

- 駕駛執照
- 護照
- 國民身分證

若由他人代為取貨，您須於結帳時將該人指定為授權取貨人，或事後於線上更新訂單資料指定。代為取貨者須出示 **本人** 的政府核發附照片身分證件，以及訂單確認信。提醒您，若未備齊上述文件，代取人將無法領取訂單，敬請事先確認。此項規定旨在維護您購買商品的安全。

## 保留期限與後續處理

訂單通常於下單後 **24 小時內** 完成備貨。備貨完成後，系統將寄送電子郵件通知，確認您的訂單已可取貨。

請於收到取貨通知後 **7 天內** 完成取貨。逾期未取的訂單，將予以取消並辦理退貨入庫，敬請於期限內前往門市領取。

若您在取貨過程中遇到任何問題，或需要延長取貨期限，請聯絡客服團隊，我們將協助您處理。門市現場的服務人員與客服團隊皆樂意為您提供協助。
