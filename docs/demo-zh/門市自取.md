# 門市自取

本文件說明門市自取的地點、取貨時效與注意事項，協助顧客規劃到店取貨流程。

## 自取地點

台北信義、台中西屯、高雄夢時代三家門市皆提供自取，**免運費**。

## 取貨時效

付款完成後，一般於 **2 小時內**（門市營業時間內）備妥，訂單狀態顯示「**可取貨**」後即可前往。請勿在狀態更新前到店，門市無法提前調貨、也無法提早交付商品。

## 保留期限

門市自取商品（非超商取貨包裹，超商包裹為到店後保留 7 天）自「可取貨」通知起於門市保留 **3 天（日曆天）**，逾期未取訂單自動取消並**全額退款**，款項依標準退款時程（**5 至 7 個工作天**）退回原付款方式。

## 取貨需知

取貨時請出示**訂單條碼**與**身分證件**。可委託他人代領，代領人需出示訂單條碼與自己的身分證件。
