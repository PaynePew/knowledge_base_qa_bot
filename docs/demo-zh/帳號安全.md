# 帳號安全

本文件說明 ACME 商店帳號的安全機制與保護措施。

## 兩步驟驗證

ACME 商店提供兩步驟驗證（2FA），於登入時要求第二道驗證，為帳號增加一層保護。啟用方式為登入後進入「帳號安全」設定，選擇「啟用兩步驟驗證」，並選定簡訊或驗證器 App（如 Authenticator）作為驗證方式，依畫面指示完成設定即可。

啟用後，每次於陌生裝置登入時，系統將要求輸入一組驗證碼。兩步驟驗證不收取任何費用。若遺失驗證裝置而無法登入，請聯絡客服，我們將於 **24 小時** 內協助確認身分並還原帳號存取權限。

## 異常登入偵測

ACME 商店的系統會持續監控帳號是否出現異常登入行為，例如來自陌生地點或裝置的登入。一旦偵測到可疑活動，系統將立即以電子郵件或簡訊通知帳號持有人，並附上登入的時間、裝置與地點資訊。

若該登入並非本人操作，建議立即透過「忘記密碼」流程更新密碼。必要時，ACME 商店會暫時鎖定帳號以防止盜用，待身分確認後解除，通常於聯絡客服後 **24 小時** 內處理完成。

## 密碼與安全建議

為保障帳號安全，建議顧客採取以下措施：

- 設定至少 **8 個字元** 並混合大小寫、數字與符號的密碼。
- 不同網站使用不同密碼，避免共用。
- 定期於「信任裝置」頁面檢視並移除不再使用的裝置。
- 不在公用電腦勾選「記住我」自動登入。

若懷疑帳號遭盜用，請立即更改密碼並移除所有信任裝置，客服將協助您完成安全設定。
