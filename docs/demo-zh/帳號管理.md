# 帳號管理

本文件說明 ACME 商店會員帳號的個人資料維護、密碼重設、電子信箱變更、偏好設定與地址簿管理，以及帳號關閉的相關規定與流程。

## 個人資料與偏好設定

登入帳號後，請點選頁面右上角的個人圖示，進入「帳號設定」頁面。在此您可以檢視並更新個人資料，包括姓名、聯絡電話與生日等基本資訊。修改完成後，請務必點擊「儲存變更」，系統才會套用更新。

「帳號設定」中的「偏好設定」分頁可調整下列項目：

- 電子報與行銷通知的訂閱狀態。
- 訂單與出貨狀態的通知方式（電子信箱或簡訊）。
- 介面顯示語言與時區。

上述偏好設定可隨時調整，且不收取任何費用。若您不希望收到行銷訊息，建議在此分頁關閉相關通知，調整將於儲存後即時生效。

## 地址簿管理

ACME 商店提供地址簿功能，方便您在結帳時快速選用常用的收件與帳單地址。請於「帳號設定」中的「地址簿」分頁進行管理：

- 新增地址：填寫收件人姓名、聯絡電話、完整地址與郵遞區號後儲存。
- 編輯地址：點選欲修改的地址旁的「編輯」按鈕，更新後儲存。
- 刪除地址：點選「刪除」並確認，即可移除不再使用的地址。
- 設定預設地址：可分別指定預設收件地址與預設帳單地址，結帳時將自動帶入。

請注意，若某筆地址正用於尚未完成的訂單，刪除後不影響該訂單已確認的配送資訊。建議定期檢查地址簿，確保收件資訊正確，以免影響出貨。

## 密碼重設

若您忘記 ACME 商店帳號的密碼，請依下列步驟重設：

1. 前往 ACME 商店首頁，點選右上角的「登入」。
2. 在登入頁面點選「忘記密碼？」連結。
3. 輸入與帳號綁定的電子信箱，並送出申請。
4. 至信箱收取密碼重設信，點擊信中的連結後，於安全頁面設定新密碼。

密碼重設信通常於數分鐘內送達，尖峰時段最長可能需要 **15 分鐘**。若遲遲未收到，請先檢查垃圾郵件或廣告信件匣，並確認輸入的電子信箱正確無誤。僅已註冊的帳號可進行密碼重設；若不確定當初使用哪個信箱，可在信箱中搜尋過往 ACME 商店的往來信件。

基於安全考量，新密碼需符合下列要求：

- 長度至少 **8 個字元**。
- 同時包含英文字母、數字與特殊符號。

密碼重設連結具有時效性，將於 **24 小時後** 失效，建議您儘速完成設定。密碼重設不收取任何費用，且可視需要重複操作。若仍有問題，ACME 商店客服團隊提供 **全年無休（24 小時）** 的線上聊天與電話協助。

## 變更電子信箱

如需變更帳號綁定的電子信箱，請先以目前的帳號密碼登入，接著依下列步驟操作：

1. 進入「帳號設定」中的「個人資訊」分頁。
2. 找到「電子信箱」欄位，點選旁邊的「編輯」按鈕。
3. 輸入新的電子信箱，並再次輸入帳號密碼以驗證身分。
4. 確認無誤後，點擊「儲存變更」。

完成後，系統將寄送一封確認信至您的新信箱，信中包含一組驗證連結，需點擊該連結才算完成變更。驗證信通常於 **5 至 10 分鐘** 內送達；若未收到，請檢查垃圾郵件匣。

請特別留意，若未在 **24 小時內** 完成驗證，基於安全考量，本次變更將自動取消，帳號仍維持原本的電子信箱。此外，新的電子信箱不得已被其他 ACME 商店帳號使用，以免發生衝突。變更電子信箱不收取任何費用，且可隨時辦理。

若您已無法存取舊信箱，且因此無法登入帳號，請直接聯絡客服，並配合完成額外的身分驗證程序。

## 關閉帳號

若您決定永久關閉 ACME 商店帳號，請登入後前往「帳號設定」，點選「關閉帳號」選項，並依畫面指示確認申請。請務必了解，帳號一經關閉，您的購買紀錄與個人資料將被永久刪除，且此動作 **無法復原**。

申請關閉帳號前，請先確認下列事項：

- 所有進行中的訂單皆已完成，且無待處理的退款或退貨；帳號關閉後將無法再行處理。
- 若仍有禮物卡餘額，請於關閉前使用完畢，否則餘額將被作廢。
- 若您訂閱了週期性服務或參加會員忠誠計畫，請另行取消，以免日後產生費用。

帳號關閉通常於申請後 **48 小時內** 完成。在此期間，若您有任何疑問或改變心意，仍可聯絡客服協助處理。關閉帳號不收取任何費用，亦無任何隱藏費用。

若您在過程中遇到問題，或有特殊情況（例如法律相關事項或個人資料隱私請求），請聯絡 ACME 商店客服團隊，我們將為您提供個別協助。
