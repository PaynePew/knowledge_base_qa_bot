# 退款與退貨

本文件說明退款申請的窗口、條件、流程與時效。

## 退款申請窗口

顧客可於**收到商品後 14 天內**（含例假日）提出退款申請，超過期限恕不受理。

## 退款條件

- 商品未經使用，且包裝完整。
- 附上原始購買憑證（電子發票或訂單編號）。
- 因**商品瑕疵或運送損壞**申請退款者，不受上述條件限制（見「損壞商品」）。

## 退款流程

1. 登入帳號，前往「我的訂單」。
2. 選擇訂單，點擊「申請退款」並填寫原因。
3. 客服於 **3 個工作天** 內完成審核。
4. 審核通過後，款項於 **5 至 7 個工作天** 內退回原付款方式。

## 退貨運費

- 非瑕疵退貨：由顧客負擔 **新台幣 80 元**，自退款金額中扣除。
- 瑕疵或運送損壞退貨：運費由 ACME 商店負擔。

ACME 商店**所有品類均不收取整新費**（restocking fee）。

## 不適用退款的情況

下列商品不提供退款，亦不受理退貨申請：

- 已下載的數位內容（電子書、軟體序號等）。
- 客製化商品（刻字、印花等）。
- 已開封的食品或衛生用品。
