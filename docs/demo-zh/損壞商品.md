# 損壞商品

本文件聚焦於損壞與瑕疵商品的界定範圍與判定原則，並說明回報期限與方式、處理選項與換貨時效。

## 回報期限與方式

收到損壞或瑕疵商品時，請於 **48 小時內** 至「我的訂單 → 回報問題」上傳商品與外包裝照片；照片需清楚顯示損壞部位，將作為審核與後續處理的依據。

## 處理選項

審核成立後可擇一處理：

- **免費換貨**：同品項換新，來回運費由 ACME 商店負擔。
- **全額退款**：含原訂單運費，退回原付款方式。

## 換貨時效

選擇換貨者，新品於審核通過後 **3 個工作天** 內寄出（以出貨時間計，非送達時間）。

## 與一般退貨的差異

損壞/瑕疵商品不受「未使用、包裝完整」的退貨條件限制，退貨運費亦由 ACME 商店負擔；詳見「退款與退貨」。
