# 帳號管理與安全

本文件說明 ACME 商店帳號的註冊、密碼、安全設定與刪除流程。

## 註冊方式

支援三種註冊方式：**Email**、**Google** 帳號、**LINE** 帳號。以第三方帳號註冊後，仍可於「帳號設定」補設密碼作為備援登入方式。

## 密碼規則

密碼長度至少 **10 碼**，且需同時包含**大寫字母、小寫字母與數字**。

## 兩步驟驗證

於「帳號設定 → 安全性」可開啟兩步驟驗證，支援兩種方式：

- 簡訊 OTP 驗證碼
- 驗證器 App（TOTP）

開啟後，每次於新裝置登入都需要輸入第二重驗證碼。

## 登入保護

連續 **5 次**登入失敗，帳號將鎖定 **30 分鐘**。鎖定期間可透過「忘記密碼」流程重設密碼提前解鎖。

## 帳號刪除

於「帳號設定」提出刪除申請後，有 **14 天猶豫期**，期滿帳號將永久刪除且無法復原。猶豫期內重新登入即可取消刪除申請。
