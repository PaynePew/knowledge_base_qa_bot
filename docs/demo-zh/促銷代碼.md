# 促銷代碼

本文件為促銷代碼的總述性說明，摘要其使用方式、使用規則、不適用範圍與逾期規定；具體條款與操作細節見以下各節。

## 使用方式

於結帳頁面的「促銷代碼」欄位輸入代碼並套用。**結帳送出後不可追加、不可補用**，請於送出訂單前確認。

## 使用規則

- 每筆訂單限用 **1 組** 促銷代碼。
- 促銷代碼可與紅利點數折抵**並用**：系統先套用代碼折扣，再以點數折抵剩餘金額。

## 不適用範圍

- **禮物卡**商品。
- **已在特價中**的商品。

## 逾期與補發

促銷代碼逾期即失效，效期**不予延長**、**不提供補發**。代碼的有效期限以活動頁面公告為準。
