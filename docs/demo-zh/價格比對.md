# 價格比對

本文件說明 ACME 商店價格比對保證的適用條件、所需證明、申請流程、排除項目與申請期限。

## 適用條件

ACME 商店致力於為顧客提供具競爭力的價格。為確保您享有最優惠的交易，我們提供價格比對保證：若您在合格的競爭零售商處發現相同商品的價格更低，可向我們提出價格比對申請，以補回差額。

商品須符合以下條件才適用價格比對：

- 商品須與競爭零售商所販售者完全相同，包含品牌、型號與顏色。
- 商品須在 ACME 商店與該競爭零售商皆有現貨，且可立即購買。
- 比對的競爭價格須為您向 ACME 商店下單時，同一國家境內於網路或實體門市公開刊登的售價。
- 競爭零售商須為合法且經授權的零售商。

請注意，價格比對僅針對商品本體售價進行比對；運費與稅費等項目若有差異，可能需要額外查核。

## 所需證明

提出價格比對申請時，請備妥可佐證競爭價格較低的證明文件，以利我們查核：

- 競爭零售商售價的截圖，或可直接連至該優惠頁面的網址連結。
- 實體門市的紙本或數位廣告。

證明文件須清楚呈現商品的品牌、型號與規格，以確認其與您所購買的商品完全一致。客服人員將依據您提供的證明，查核競爭價格是否屬實並符合所有適用條件。

## 申請流程

請於購買後 **14 天內** 透過 ACME 商店網站、電子郵件或電話聯絡客服團隊，提出價格比對申請。申請流程如下：

1. 聯絡客服團隊，並提供購買訂單資訊。
2. 附上可佐證競爭價格的證明文件（截圖、連結或廣告）。
3. 客服人員將查核競爭價格，並確認其符合所有適用條件。
4. 申請核准後，差額將以您原本的付款方式退回。
5. 退款一般於 **5 至 7 個工作天** 內完成。

ACME 商店不會因處理價格比對申請而向顧客收取任何額外費用。若涉及運費或稅費差異等情況，可能需要額外查核，處理時間將因此延長。

## 排除項目

下列情況不適用價格比對：

- 限時促銷活動，例如快閃特賣、限時搶購等。
- 出清特賣商品。
- 組合優惠（搭售）商品。
- 會員專屬價格或折扣會員俱樂部價格。
- 拍賣型網站的價格。
- 未經授權零售商的價格。

## 注意事項

價格比對保證僅適用於符合上述所有條件的商品與競爭價格。ACME 商店保留依個案情況拒絕價格比對申請的權利，惟我們承諾對每一筆申請進行公平且合理的評估，確保您在 ACME 商店的購物體驗既實惠又愉快。
