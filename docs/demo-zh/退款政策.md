# 退款政策

本文件說明顧客申請退款的相關規定與流程。

## 退款申請窗口

顧客可在收到商品後 **14 天內** 提出退款申請。超過期限的退款申請恕不受理。

## 退款條件

退款需符合以下條件：

- 商品未經使用，且包裝完整。
- 退款申請須附上原始購買憑證（發票或訂單編號）。
- 因商品瑕疵或運送損壞而申請退款者，不受上述限制。

## 退款流程

1. 登入帳號，前往「我的訂單」頁面。
2. 選擇需要退款的訂單，點擊「申請退款」。
3. 填寫退款原因並上傳相關照片（若有商品瑕疵）。
4. 客服人員將於 **3 個工作天** 內審核您的申請。
5. 審核通過後，款項將於 **5–7 個工作天** 內退回原付款帳戶。

## 不適用退款的情況

下列商品不在退款範圍之內：

- 數位內容（電子書、軟體序號等）一經下載即無法退款。
- 客製化商品（刻字、印花等）。
- 已開封的食品或衛生用品。
