# Order Management

This document covers cancelling, modifying, and looking up orders.

## Cancelling an Order

While an order is "**Processing**" (not yet dispatched), you can cancel it yourself under My Orders. **Orders cannot be cancelled after dispatch** — use the returns process instead (see "Refunds & Returns").

## Modifying an Order

- **Shipping address**: editable under My Orders any time before dispatch.
- **Payment method**: self-service changes within **1 hour** of placing the order; after that, cancel and reorder.
- Items cannot be modified — cancel and place a new order.

## Order History & Invoices

Order records are kept for **5 years**. Look up past orders and download e-invoices under My Orders.
