# Promo Codes

This document covers how promo codes work and their restrictions.

## How to Use

Enter the code in the **Promo code** field at checkout and apply it. Codes **cannot be added after checkout** — confirm before submitting the order.

## Rules

- **One promo code per order**.
- Promo codes **stack with reward points**: the code discount is applied first, then points can cover part of the remainder.

## Exclusions

- **Gift card** purchases.
- Items **already on sale**.

## Promo Code Expiry

Expired codes are invalid and **not reissued**. Validity dates are stated on each campaign page.
