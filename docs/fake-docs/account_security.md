# Account Security

## Two-Factor Authentication

Customers can enable two-factor authentication from the security section of profile settings, choosing either an authenticator app or a code sent by text message. Once enabled, a second factor is required only on a new device or after thirty days on a trusted device, so routine sign-ins on a personal computer are not interrupted.

## Suspicious Activity

If sign-in attempts from an unrecognised location are detected, the account is temporarily locked and a verification email is sent to the address on file. The lock clears as soon as the customer confirms the activity through that email or resets the password from a trusted device.

## Managing Trusted Devices

A list of trusted devices appears in the security section, where any device can be removed individually or all devices can be signed out at once. Removing a device forces a fresh two-factor challenge the next time that device signs in.
