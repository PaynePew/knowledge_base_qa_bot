# Order Management

## Cancel an Order

An order can be cancelled directly from the order history page as long as it has not yet entered the packing stage. Once packing begins the cancel button disappears and the customer must instead refuse delivery or start a return after the parcel arrives. Cancelling a multi-item order cancels every line at once; partial cancellation of individual items is not supported.

## Change Shipping Address

The shipping address can be edited until the order reaches the packing stage, after which the destination is locked. If an address must change after that point, support can sometimes intercept the parcel with the carrier for a five-dollar redirection fee, though interception is not guaranteed once the parcel is in transit.

## Combine Orders

Two separate orders placed within the same hour to the same address can be combined into one shipment on request, which saves a second shipping fee. Combined orders ship only when every item across both orders is in stock, so a backordered item delays the entire combined shipment.

## Backordered Items

When an item is backordered, the order page shows an estimated restock date and the customer is charged only when that item finally ships. A backordered item can be removed from an order at any time before it ships, and removing it does not affect the rest of the order.
