# Promotional Codes

## Applying a Code

A promotional code is entered in the dedicated field on the checkout page, and only one code can be applied per order. A code that fails to apply usually has either an unmet minimum order value or a category restriction that excludes one or more items in the cart.

## Stacking Rules

Promotional codes cannot be stacked with each other, and a percentage-off code cannot be combined with loyalty point redemption on the same order. A free-shipping code is the one exception and may be combined with a single percentage-off code because the two discounts apply to different parts of the total.
